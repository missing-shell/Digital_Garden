| 转义序列       | 含义            |
| ---------- | ------------- |
| \\         | \ 字符          |
| \'         | ' 字符          |
| \"         | " 字符          |
| \?         | ? 字符          |
| \a         | 警报铃声          |
| \b         | 退格键           |
| \f         | 换页符           |
| \n         | 换行符           |
| \r         | 回车            |
| \t         | 水平制表符         |
| \v         | 垂直制表符         |
| \ooo       | 一到三位的八进制数     |
| \xhh . . . | 一个或多个数字的十六进制数 |
